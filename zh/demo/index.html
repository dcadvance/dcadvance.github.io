# 大图

大图不要存放在站点下面，github 可存储空间有限，资源占用过量会导致站点无法更新。

使用 腾讯云的COS服务 (opens new window) 来存储图片。

使用 COSBrowser (opens new window) 工具来上传图片。

使用 数据万象 (opens new window) 服务来提供不同的图片尺寸。

图片格式，除非要求半透明，否则都要变更为 jpg, 限制为 200kb 以下。

图片压缩可以使用在线工具: https://squoosh.xiaoxili.com/ (opens new window)
